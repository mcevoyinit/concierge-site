How it works

Three quick steps you do yourself — the installer handles everything else.

1

Copy & run the command

Open Terminal (press ⌘ Space, type “Terminal”, hit Enter), paste the line from the top of this page, and press Enter. It installs Claude Code and the skill pack.

Terminal
$ curl -fsSL …/install.sh | bash

2

Sign in to Claude

Open a new Terminal window (so it sees Claude), type claude and press Enter. A browser window opens — sign in with your Claude account (the same one as claude.ai).

3

Authorize Gmail & Calendar (once)

Powers your morning brief and email digest. Claude Code can't run Google's sign-in itself, so you authorize once in your browser at claude.ai → Connectors. After that it all runs right here in Claude Code — you never go back.

🩺 Mac health

Tells you what’s really eating your Mac’s memory — and what’s safe to close.

🎙️ Biographer

An agent that interviews you and builds a private profile over time.

Private by design

  • It’s a clean-room install — it sets up the tools on your Mac with your accounts. No one else’s data is ever included.
  • Your keys and sign-ins stay on your machine. Nothing is sent to us.
  • The install script is open to read before you run it.
  • Re-run the command any time to update — it backs up your setup first and never overwrites your settings.
